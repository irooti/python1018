*값형식(Value type):값을 복사해서 전달(int, float, bool...) 
pass by value, call by value 
(Java, C#...)
int x =5;
int y = x;
y = 6; 
stack(입력으로 역순으로 처리하는 구조, 메모리 공간) 
------
x(5)
y(6)
------

*참조형식: 클래스를 통해서 객체를 사용하는 형식(파이썬은 이 형식만 제공)
pass by reference, call by reference 
Demo d = new Demo();
Demo d2 = d; (d의 참조만 복사해서 전달) 

heap(연속된 메모리 공간)
--------
객체 <------------ d (본명)
       <------------ d2 (별명) 
--------
